# Desi Cart App

Desi Cart is an online shopping application developed using Flutter, Python (Flask), and MySQL.

## Technologies Used

- Flutter
- Python
- Flask
- MySQL

## Features

- User Registration & Login
- Product Management
- Shopping Cart
- Order Management
- Admin Panel
- Seller Panel
- delivery_staff Panel

## Project Structure

```
desi-cart-app/
├── Backend/
├── Frontend/
│   └── my_app/
└── shopping_system.sql
```

## How to Run

### Backend

```bash
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
flutter pub get
flutter run
```

## Database

Import `shopping_system.sql` into MySQL before running the project.
